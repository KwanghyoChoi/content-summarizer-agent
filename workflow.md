# Claude Code 초기 프롬프트

이 파일은 Claude Code에서 프로젝트를 시작할 때 첫 번째 메시지로 입력하세요.

---

## 프로젝트 초기화 프롬프트

```
이 프로젝트는 YouTube, PDF, 웹페이지에서 정확한 요약 노트를 생성하는 시스템이야.

### 핵심 원칙 (절대 준수)
1. **원문에 있는 내용만** 요약에 포함 - 추론/추가/확장 금지
2. **모든 내용에 출처 표기** - [HH:MM:SS] 또는 [p.N]
3. **불확실한 내용**은 "원문에서 명확하지 않음"으로 표기
4. **원문 전체를 항상 보존**하고 요약과 함께 제공

### 워크플로우
1. 추출: python main.py로 원본 텍스트 추출
2. 노트 생성: templates/ 폴더의 프롬프트 참조하여 4가지 형식 생성
   - 자세한 노트 (detailed): 전체 내용 + 계층 구조
   - 핵심 노트 (essence): 5-10개 핵심 포인트
   - 쉬운 노트 (easy): 입문자용 3-5개 최핵심
   - 마인드맵 (mindmap): Mermaid 시각화

### 사용 명령
- YouTube: python main.py --youtube "URL"
- PDF: python main.py --pdf "경로"
- 웹: python main.py --web "URL"

CLAUDE.md 파일에 상세 규칙이 있으니 참조해.
```

---

## 일반 요약 요청 프롬프트 예시

### YouTube 영상 요약
```
이 YouTube 영상을 요약해줘: [URL]

1. 먼저 main.py로 자막을 추출해줘
2. templates/detailed.md를 참조해서 자세한 노트 생성
3. templates/essence.md를 참조해서 핵심 노트 생성
4. 모든 노트를 output 폴더에 저장
```

### PDF 문서 요약
```
이 PDF를 요약해줘: [파일경로 또는 첨부]

1. extractors/pdf.py로 텍스트 추출
2. 4가지 형식 노트 모두 생성
3. 원문에 있는 내용만, 타임스탬프 대신 페이지 번호 사용
```

### 웹페이지 요약
```
이 웹페이지를 요약해줘: [URL]

1. extractors/web.py로 본문 추출
2. 핵심 노트와 마인드맵 생성
3. 출처 URL과 섹션 위치 표기
```
